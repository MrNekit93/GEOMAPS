# Инструкция по запуску проекта GEOMAPS

## Описание
Приложение для сравнения алгоритмов поиска маршрутов (Дейкстра и A*) на карте Беларуси.

## Требования
- Python 3.8+ (для сервера)
- Android Studio (для клиента)
- Файл `belarus_graph.json` в папке `server/`

## Запуск сервера

1. Перейдите в папку server:
```bash
cd D:\GEOMAPS\server
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Запустите сервер (IP замените на ваш):
```bash
uvicorn server:app --host 0.0.0.0 --port 8000
```

Сервер будет доступен по адресу: http://192.168.100.197:8000

## Запуск Android-клиента

1. Откройте проект `D:\GEOMAPS` в Android Studio
2. Убедитесь, что в `RouteService.kt` указан правильный IP сервера:
```kotlin
private val baseUrl = "http://192.168.100.197:8000"
```
3. Запустите приложение на устройстве или эмуляторе

## Использование

1. **Выбор точек**: долгое нажатие на карте (зеленый - старт, красный - финиш)
2. **Поиск мест**: введите название города в поле поиска
3. **Алгоритмы**: 
   - "Дейкстра" - построение маршрута алгоритмом Дейкстры
   - "A*" - построение маршрута алгоритмом A*
   - "Сравнить" - сравнение обоих алгоритмов
4. **Типы дорог**:
   - "Все дороги" - использование всех дорог
   - "Магистрали" - только магистрали (motorway, trunk)
5. **Скорость**: переключение между 60 км/ч и 110 км/ч

## Файлы проекта

### Сервер (Python/FastAPI)
- `server/server.py` - основной сервер
- `server/requirements.txt` - зависимости Python

### Клиент (Android/Kotlin)
- `app/src/main/java/com/example/geomaps/MainActivity.kt` - главный экран
- `app/src/main/java/com/example/geomaps/RouteService.kt` - API клиент
- `app/src/main/res/layout/activity_main.xml` - разметка
- `app/src/main/AndroidManifest.xml` - манифест
- `app/build.gradle.kts` - конфигурация сборки

## API Эндпоинты

1. `GET /health` - проверка статуса
2. `POST /route/{algorithm}` - построение маршрута (algorithm: dijkstra или a_star)
3. `POST /route/compare` - сравнение алгоритмов

## Примечание
- Файл `belarus_graph.json` должен находиться в папке `server/`
- Приложение требует доступа к интернету
- Сервер должен быть доступен с устройства по IP-адресу
